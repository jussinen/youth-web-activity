Local Web Server

  1. Create a folder on your desktop
  2. Add a file called readme.txt
  3. Open Command Prompt
  4. Run http-server -p 123
  5. Visit http://localhost:123/readme.txt
Point out that we just turned our computer into a server!

Is Today a Holiday?

  • Add a holidays.json file
  • Use fetch() in JavaScript
  • If today matches, show a Bootstrap callout
fetch('holidays.json')
  .then(res => res.json())
  .then(holidays => {
    const today = new Date().toISOString().split('T')[0];
    const match = holidays.find(h => h.date === today);
    if (match) {
      const el = document.createElement('div');
      el.className = 'alert alert-info';
      el.innerText = `ðŸŽ‰ It's ${match.name} today!`;

Search Birthdays with SQL

  • We use sql.js to run SQLite in the browser
  • Type a name, see their birthday from a local DB
  • All happens without a server!
const db = new SQL.Database();
db.run("CREATE TABLE people (name TEXT, birthday TEXT);");
db.run("INSERT INTO people VALUES ('Alice', '2005-06-15');");

function find(name) {
  const stmt = db.prepare("SELECT birthday FROM people WHERE name = ?");
  stmt.bind([name]);
  while (stmt.step()) {
    console.log(stmt.getAsObject()); // { birthday: '2005-06-15' }
  }
  stmt.free();
}
Point out that this simulates what a real database would do on the backend.
