Style with HTML

You can add styles directly using the style attribute.

<h1 style="color: red;">Welcome!</h1>

This changes the color of the text to red.

This is called "inline styling" and is a good way to experiment.

Use a CSS Class

You can write a style once and apply it to many elements using a class.

<style>
  .highlight {
    background-color: yellow;
    font-weight: bold;
  }
</style>

<p class="highlight">This is important!</p>

This separates your content from the style.

Explain the importance of reusability. Highlight how styles go in the <head> or top of the page.

What Are Templates?

Templates are blueprints for pages.

  • They let you show dynamic content (like different names or dates)
  • You don‚Äôt copy and paste whole pages ‚Äî just fill in the blanks
  • Templates save time and keep code clean
Ask: ‚ÄúWhat if I had to make a new HTML file for each person‚Äôs name?‚Äù Templates automate that!

Is Today a Holiday?

Use a JSON file to list holidays, and show an alert if today matches.

holidays.json:

[
  { "date": "2025-12-25", "name": "Christmas" },
  { "date": "2025-01-01", "name": "New Year‚Äôs Day" }
]

main.js:

document.addEventListener('DOMContentLoaded', () => {
  fetch('holidays.json')
    .then(res => res.json())
    .then(holidays => {
      const today = new Date().toISOString().split('T')[0];
      const match = holidays.find(h => h.date === today);
      if (match) {
        const alert = document.createElement('div');
        alert.className = 'alert alert-info';
        alert.role = 'alert';
        alert.innerText = `üéâ Today is ${match.name}!`;
        document.body.prepend(alert);
      }
    });
});

Include Bootstrap for alert styling, and make sure this JS is linked at the bottom of your HTML:

<script src="main.js"></script>

What is a Database?

  • A place to store lots of structured information
  • Like a giant spreadsheet with rows and columns
  • Used to quickly find and update data

Examples:

  • Usernames and passwords
  • Product prices
  • Birthdays and appointments
Compare to Excel or Google Sheets. Each row = a record, each column = a field.

What is SQL?

  • SQL = Structured Query Language
  • It's the language we use to talk to a database
  • We can ask it questions like:
SELECT * FROM people;
SELECT birthday FROM people WHERE name = 'Alice';

You get rows back with the answer!

It's like asking a question in English ‚Äî ‚ÄúGive me all people‚Äù or ‚ÄúWhat‚Äôs Bob‚Äôs birthday?‚Äù

Check if Today is Someone‚Äôs Birthday

Add this inside your script, after setting up the database:

db.run("INSERT INTO people VALUES ('Alice', '2005-05-19');");
db.run("INSERT INTO people VALUES ('Bob', '2006-12-22');");

const today = new Date().toISOString().slice(5, 10); // MM-DD

const result = db.exec(
  "SELECT name FROM people WHERE substr(birthday, 6, 5) = ?",
  [today]
);

if (result.length > 0) {
  const names = result[0].values.map(row => row[0]).join(", ");
  const alert = document.createElement("div");
  alert.className = "alert alert-success";
  alert.role = "alert";
  alert.innerText = `üéâ Today is ${names}'s birthday!`;
  document.body.prepend(alert);
}
